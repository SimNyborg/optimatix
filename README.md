# Optimatix – Hjemmeside

Flask-baseret hjemmeside for Optimatix med support for dansk og engelsk.

## SEO og Google Search Console

**Google Search Console** er Googles officielle værktøj til at overvåge og optimere en hjemmesides tilstedeværelse i søgeresultater. Det viser, hvordan Google finder, crawler og indekserer hjemmesiden, samt hvilke søgeord der driver trafik, hvor mange visninger og klik siden får. Værktøjet hjælper med at opdage tekniske SEO-fejl, validere sitemap.xml og robots.txt, og skal ses som en integral del af SEO-strategi uden betalt annoncering. Det er gratis og vedligeholdes af Google.

For denne hjemmeside skal Search Console konfigureres til at overvåge `https://optimatix.dk` og validere sitemappet via `/sitemap.xml`.
